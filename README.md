## What This Is

An interactive operations dashboard for CNC Electric Pakistan's aggressive backlink building campaign. The dashboard manages daily tasks for 3 team members, generates AI content via Claude API, and tracks progress toward DR 50.

**Current:** DR 1.8 | 27 referring domains | 220 backlinks | 1,153 monthly traffic  
**Target:** DR 50 | 700+ referring domains | 3,000+ backlinks  
**Timeline:** 3 months (April - June 2026)

## Dashboard Tabs

| Tab | Who | What It Does |
|-----|-----|-------------|
| **Hanan Tasks** | Executor | Daily calendar with step-by-step manual tasks (create profiles, post answers, send emails) |
| **Bilal AI Engine** | Content | One-click AI generation — Quora answers, outreach emails, Medium articles, Pinterest pins |
| **Muddasir Backlog** | Technical | Claude Code task queue — calculators, analyzers, infographics, data reports |

## AI Features

The dashboard integrates directly with the **Anthropic Claude API**:

- **Quora Answer Generator**: Select from 20 mapped questions → generates 300-word expert answer with correct deep link
- **Email Generator**: Select type + fill recipient → generates personalized outreach email
- **Content Multiplier**: Generate Medium articles, Pinterest pins, Google Business posts, YouTube descriptions
- **AI Chatbot**: Full project context loaded — ask it to write any content, analyze strategy, or check progress

## Tech Stack

- React (JSX)
- Anthropic Claude API (claude-sonnet-4-20250514)
- Persistent storage for progress tracking
- No external dependencies beyond React + Recharts
