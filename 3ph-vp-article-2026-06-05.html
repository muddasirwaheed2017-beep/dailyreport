1. What is a 3-phase voltage protector?

A 3-phase voltage protector continuously monitors all 3 phases (R, Y, B) and cuts the load if any of the following occur:

  • Any phase voltage exits 170-265V safe band
  • Phase asymmetry exceeds 10% (one phase ≠ others)
  • Phase failure (any phase drops out)
  • Wrong phase rotation (will not allow start)
  • Neutral failure (in 4-wire systems)

It restores power automatically once all 3 phases return to normal for a programmable delay (typically 5-30 seconds).

3. Sizing rules for Pakistani 3-phase loads

  1. Total your 3-phase load in kW (sum all motors, ACs, equipment).
  2. Convert to amps: kW × 1000 / (1.732 × 380V × 0.85 PF) = continuous current per leg.
  3. Add 25% safety margin.
  4. Pick the next standard catalogue rating: 25A, 40A, 63A, 100A, 125A, 200A.

4. Installation in a 3-phase DB box

  1. Power off at the upstream main breaker. Verify zero voltage on each phase with a tester.
  2. Mount the protector on DIN rail or wall depending on model size.
  3. Wire R, Y, B phases + Neutral from the grid feed into the protector's IN terminals.
  4. Wire R, Y, B phases + Neutral from the protector's OUT terminals to the downstream MCB row or motor contactor.
  5. Connect earth to the protector's earth terminal.
  6. Power on. Set parameters:
    - Under-voltage trip: typically 195V per phase
    - Over-voltage trip: typically 265V per phase
    - Asymmetry threshold: 10%
    - Restoration delay: 10 seconds
  7. Test: simulate phase loss by switching off one upstream MCB; protector should trip within 100-300 ms.

6. FAQ

Cheapest 3-phase voltage protector in Pakistan 2026?

Do I need a 3-phase protector if I have a phase failure relay?

A phase failure relay only detects phase loss. A 3-phase voltage protector adds over-voltage, under-voltage, asymmetry, and rotation protection. Best practice: use both, or use a 3-phase protector that includes phase failure detection.

Will it protect a 3-phase solar inverter?

Yes — install upstream of the inverter's AC input. Protects against grid events that could damage the inverter.

Is the WiFi smart variant worth +PKR 6,000?

For factories where downtime costs money: yes — app alerts, runtime logs, scheduled tests. For typical residential 3-phase: standard variant is enough.

Will CNC YC3VP-series work with Inverex/Knox/Solis solar inverters?

Yes — it's grid-side protection, inverter brand-agnostic. Install between WAPDA feed and your DB/inverter input.
