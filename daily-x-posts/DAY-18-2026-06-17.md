**Tweet 2 (education):**
There are 4 accuracy classes for 3-phase energy meters in Pakistan (IEC 62053-21/22):

• Class 2 → ±2% error → Rs 2,800–4,500
• Class 1 → ±1% error → Rs 4,500–28,000
• Class 0.5S → ±0.5% error → Rs 18,000–65,000

On a Rs 500K/month industrial bill, Class 2 vs Class 1 = Rs 5,000/month mystery "error" that always goes the DISCo's way.

---

**Tweet 3 (specifics):**
If you have solar net metering in Pakistan, NEPRA rules say your check meter must be:

✅ Class 0.5S (IEC 62053-22)
✅ Bi-directional (separate import + export registers)
✅ Your own — not the DISCo's sealed meter

Without it, you cannot dispute short-credited export kWh in a NEPRA hearing.

**Tweet 4 (mistake):**
The CT ratio mistake that costs factories every month:

Wrong: "I'll buy a cheap 100A direct-connect meter for my 200A feeder"
Right: CT-operated meter + 300/5A CT (size = Full Load × 1.2 ÷ 5)

Direct-connect meter overloaded → inaccurate for months before anyone notices. CT accuracy class must match meter class.
