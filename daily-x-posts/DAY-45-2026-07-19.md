## Post 3 — Thread (load-shedding PLC backup angle)

Standard 24VDC PSU loses output in <10ms when WAPDA cuts.

Result:
❌ PLC resets
❌ Active write to PLC program corrupted
❌ All pilot lamps off
❌ VFD run command drops → motors coast to stop

Pakistan solution: UPS-function DIN rail PSU
→ Integrated battery charger + auto-switchover in <20ms
→ 7Ah SLA battery → ~35 min backup at 5A load
→ Price: Rs. 12,000–16,000 (PSU only) + Rs. 2,500 (7Ah SLA)
