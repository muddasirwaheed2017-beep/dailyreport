## Post 2 — The Fatal Mistake Pakistani Electricians Make

Pakistan electricians set the overload relay dial to MAXIMUM range.

5 HP motor = 8.5A FLC.
Relay range: 7–10A.
Dial set to: 10A.

Result: motor can run at 115% overload continuously with ZERO protection.

Set dial to ACTUAL motor FLC. Not to relay max.

## Post 3 — Trip Class Explained in One Table

Trip Class confusion costs Pakistani motors every summer:

🔴 Class 10 = trips in ≤10 sec at 7.2× FLC
⚠️ Problem: tubewell DOL start takes 5–8 sec. Class 10 trips on normal start.

🟢 Class 20 = trips in ≤20 sec
✅ Default choice for Pakistan: tubewells, compressors, conveyors, HVAC

🔵 Class 30 = trips in ≤30 sec
Only for high-inertia loads (centrifuge, large fan)

When in doubt → choose Class 20.

## Post 4 — WAPDA Voltage Problem

Your WAPDA supply runs at 360–370V.
Motor rated at 380V.

Motor current INCREASES when voltage drops.
At 370V instead of 380V → FLC rises ~2.7%.

5 HP motor at 380V: 8.5A FLC
Same motor at 360V: ~9.0A FLC

Your relay set to 8.5A will nuisance-trip every WAPDA dip afternoon.

Fix: set relay 5–8% ABOVE nameplate FLC in Pakistan.
