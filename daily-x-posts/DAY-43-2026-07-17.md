## Tweet 2 (Emergency Stop NC Contact Rule)

🚨 Pakistan Panel Safety Rule Most Builders Get Wrong:

E-Stop wired through Normally Open (NO) contact = machine KEEPS RUNNING if wiring breaks.

E-Stop must use NC contact in series with contactor coil.

Broken wire → contact opens → machine STOPS = fail-safe.

Pakistan labour inspection checks this. So do EU buyer audits.

Rs. 150 extra for 1NC+1NO E-stop. Worth it every time.

## Tweet 3 (PLC AUTO/MANUAL Selector Wiring)

Pakistan PLC Panel Standard — 3-Position Selector Switch Wiring:

LEFT = MANUAL → 24VDC → PLC DI #1
CENTRE = OFF → both DIs off (safe state)
RIGHT = AUTO → 24VDC → PLC DI #2

Never mix 24VDC and 220VAC on the same common terminal.

Most common Pakistan PLC panel wiring error — destroys PLC input cards.

Full wiring guide at cncelectric.pk/blogs/guides

## Tweet 4 (22mm vs 30mm — Pakistan Standard)

Pakistan Panel Cutout Standard: 22mm or 30mm?

✅ 22mm = Pakistan MCC standard
→ ABB/Schneider/LS/Chint/Delixi — all 22mm
→ DOL starters, VFD panels, PLC panels

30mm = heavy duty outdoor / gloved operator only

22mm and 30mm components are NOT interchangeable.
Order the wrong size = Rs. 2,000–8,000 metalwork modification.

## Tweet 5 (Standalone — Common Mistake)

Most skipped Pakistan panel step: the 22mm seal ring (O-ring).

Every 22mm pushbutton comes with a rubber seal ring for the panel cutout.

Pakistan panel builders discard it. Panel is now IP40 at every cutout — not IP65.

Rain, cotton lint, cement dust → inside the panel within weeks.

Rs. 40–80 rubber ring. Don't throw it away.
