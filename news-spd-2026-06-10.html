Quick answer

"SPD" stands for Surge Protection Device. The IEC 61643 family defines three classes: Type 1 (handles direct lightning strike, installed at building entry), Type 2 (handles indirect surge from grid, installed at main DB), Type 3 (handles residual surge at sensitive equipment, installed inside the appliance). For Pakistani residential, Type 2 is the default. NEPRA 2026 solar net-metering now requires Type 2 on both AC and DC sides per IEC 61643-31.

AC SPD vs DC SPD — the NEPRA 2026 distinction

IEC 61643-11 covers AC-side SPDs; IEC 61643-31 (new 2018 standard) covers DC-side SPDs for photovoltaic systems. NEPRA's 2026 net-metering checklist requires both: Type 2 AC SPD on the inverter output side AND Type 1+2 DC SPD on the PV string side (1000V DC or 1500V DC rated depending on system topology).

  • AC SPDs: 230V / 400V rated, MOV-based, replaces on indicator change
  • DC SPDs: 600V / 1000V / 1500V rated, MOV + spark gap hybrid, different terminal layout
  • Using AC SPD on DC side is a critical safety failure — MOV ages much faster and can fail short-circuit, starting a DC arc fire

Installation sequence in Pakistani DB

  1. Incoming from DISCO meter
  2. Main isolator switch
  3. Type 2 SPD (or combined Type 1+2 if exposed building)
  4. RCCB or RCBO main
  5. MCBs to sub-circuits
  6. Type 3 SPD at sensitive equipment if needed

SPD indicator window — when to replace

Every SPD has a status indicator (green / red flag in a small window). When SPD has absorbed multiple surges over its lifetime, the MOV degrades and the indicator turns red. Replace immediately — a degraded SPD can fail short-circuit and trip your main breaker (or worse, fail open and offer no protection). Pakistani electrical inspection checks SPD indicator status on commercial + solar installations.

Six common Pakistani SPD mistakes

  1. Using AC SPD on DC PV string (critical safety failure)
  2. Skipping Type 1 on buildings with rooftop solar / exposed lightning risk
  3. Not earthing the SPD — must have dedicated earth bond < 50 cm
  4. Connecting SPD downstream of MCB instead of upstream (MCB trips before SPD activates)
  5. Not replacing on indicator change (MOV degraded = no protection)
  6. Buying Type 3 only without Type 2 upstream (Type 3 alone has insufficient discharge capacity)
