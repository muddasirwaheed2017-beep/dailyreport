NEPRA 2026 solar + lightning protection rules

  • Solar rooftop systems > 5 kW: Type 1+2 combined SPD on inverter AC output mandatory per IEC 61643-31 + IEC 61643-11
  • Solar PV string side: Type 1+2 DC SPD (IEC 61643-31) on DC bus + DC isolator combination
  • Building lightning down conductor: ground impedance < 10 Ω per IEC 62305-3
  • Telecom tower + tall building: Type 1 mandatory at building entry + Type 2 at every distribution panel
  • Indicator replacement: SPD with status indicator window MUST be replaced when MOV degrades (visible red flag) — NEPRA + Pakistani inspectorate check

Pakistani 2026 install sequence (lightning to load)

  1. External lightning rod + down conductor on roof
  2. Earth bus + dedicated lightning earth rod (< 10 Ω resistance)
  3. Building entry — Type 1 SPD on incoming feeder (after main isolator)
  4. Main MCCB (overcurrent protection)
  5. Main DB — Type 2 SPD before RCCB / RCBO main
  6. Sub-circuit MCBs
  7. Sensitive equipment — Type 3 SPD at socket or inside equipment cabinet
  8. Each SPD chassis + neutral + protective earth bonded to building earth via < 50 cm bond strap

Six mistakes Pakistani surge arrester buyers make

  1. Skipping Type 1 on buildings with external lightning rod — the lightning current bypasses Type 2 and burns through downstream equipment
  2. Using Type 3 alone without Type 2 upstream — Type 3 alone has insufficient discharge capacity
  3. Installing AC-rated SPD on solar DC bus (critical safety failure — MOV ages much faster on DC)
  4. Not replacing SPD when indicator flag turns red — degraded MOV provides no protection
  5. Skipping earth bond on SPD chassis — the protective discharge path goes nowhere
  6. Connecting SPD downstream of MCB instead of upstream — MCB trips before SPD activates
