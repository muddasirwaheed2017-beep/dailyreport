What is a solar combiner box (and which type you need)

A solar combiner box is a sealed DC distribution enclosure containing: DC string fuses (one per PV string), a DC isolator, a Type 1+2 DC SPD, and bus bars consolidating all string outputs into a single feeder to the inverter. The combiner protects each string independently against backflow, fault current, and surge events. Pakistani residential solar systems typically need a 4-8 string combiner; commercial / industrial systems run 16-32 strings.

Six mistakes Pakistani solar installers make

  1. Wrong DC fuse class. Using gG fuses instead of gPV class. gPV is specifically rated for DC PV inrush + arc characteristics; gG fails open at the wrong current.
  2. Under-rated voltage class. Installing 600V combiner on a 1000V system. Insulation breakdown + arc fault risk.
  3. Skipping the DC SPD. Rooftop combiners lose surge protection → lightning destroys inverter input. NEPRA 2026 requires Type 1+2 DC SPD on rooftop combiners.
  4. IP44 enclosure on rooftop. Should be IP65+ for direct sun + monsoon. PKR 5,000-15,000 saving turns into PKR 50,000+ retrofit cost.
  5. Mixing tier 5 unbranded fuses in tier 2-3 combiner. The cheapest component sets the system reliability. Match component tier.
  6. Skipping earth bond. Floating bus = arc fault risk + Pakistani inspection failure. Continuous earth bond mandatory per IEC 60364.
