How to add to your Shopify store
1
Go to your Shopify Theme Code
Online Store → Themes → Edit Code → Open theme.liquid
2
Find the closing body tag
</body>
3
Paste the chatbot script just before it
Use cnc_claude_chatbot_shopify.html for the AI version, or cnc_rulebased_chatbot_shopify.html for no-AI. Copy everything inside the file and paste it before </body>.
4
Set your API key via the dashboard
Open CNC_Analytics_Dashboard.html → Settings tab → Enter your Anthropic API key. The chatbot reads it from your browser's local storage automatically.
5
Watch sessions appear in your dashboard
Open the Analytics Dashboard in another tab. Every customer conversation will show up in Live Sessions within seconds.
