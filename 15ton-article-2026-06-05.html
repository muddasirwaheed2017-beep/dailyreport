3. Why these specific picks

16A MCB (not 13A, not 20A)

13A MCB nuisance-trips during compressor inrush. 20A MCB protects against overload but won't trip on partial short-circuit involving 14A continuous draw. 16A C-curve is the goldilocks pick: catches actual faults while ignoring inrush.

C-curve (not B, not D)

B-curve trips at 3-5× rated = 48-80A; 1.5-ton AC inrush hits 60A. False trips guaranteed. C-curve trips at 5-10× = 80-160A — handles inrush cleanly. D-curve is for transformers and welders, unnecessary for ACs.

Type A RCBO (not Type AC)

Modern Pakistani inverter ACs contain DC-leakage-capable electronics. Type AC RCBOs miss DC leakage and trip unreliably. Type A (which detects both AC and pulsating DC) catches the modern AC's actual leak modes. PEC code increasingly mandates Type A for sockets feeding electronics.

32A voltage protector (not 16A)

1.5-ton AC continuous draw is 7-9A, but the voltage protector needs to handle compressor inrush of 40A briefly. A 32A-rated protector handles inrush cleanly. 16A protector trips on every AC startup.

4. Wiring sequence

  1. Power off at the main DB-box.
  2. Run dedicated 2.5 mm² copper from DB-box to AC location (≤25m run typical residential).
  3. In the DB-box: voltage protector → MCB → RCBO → outgoing live.
  4. Terminate at the AC outdoor unit junction box with 16A IP44 industrial plug (optional) or direct hardwired.
  5. Earth: bond AC outdoor unit chassis to building earth.
  6. Test: power on, AC starts; verify voltage protector LED is green; verify RCBO test button trips correctly.

Can I just use a single RCBO without separate MCB?

Yes — RCBO is MCB + RCCB in one device. The separate MCB is redundant if you have RCBO. For pure single-breaker simplicity: just use the YCB10 RCBO 16A Type A.

What's the cheapest acceptable protection?

Single 16A MCB + voltage protector at PKR 3,500. Skip RCBO and AC SPD if budget is extreme. You lose earth-leakage and surge protection but keep overload + short-circuit + voltage event protection.

Does the AC need a 13A or 16A breaker?

16A C-curve is recommended. 13A is too tight for 1.5-ton AC inrush. 20A is too loose.

1.5 ton vs 1 ton AC protection — same components?
